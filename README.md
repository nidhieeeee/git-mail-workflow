# Git Mail Workflow

This project demonstrates a CI/CD workflow using GitHub Actions that sends an email notification when a Pull Request is merged into the `main` branch.

## Workflow

1. Developer creates a branch
2. Opens a Pull Request
3. Pull Request is merged into `main`
4. GitHub Actions triggers
5. Brevo API sends an email notification

## Technologies Used

- GitHub Actions
- Brevo Email API
- CI/CD pipeline
